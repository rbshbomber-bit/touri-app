# 🐰 토우리 (Touri) 브랜드 가이드

> _"언제나 네 편이에요.
> 네 마음을 이해해주고,
> 너의 하루를 더 예쁘고 의미있게 만들어주는
> 따뜻한 다이어리 친구예요."_
>
> — 캐릭터 시트 원문

## 1. 캐릭터 페르소나

### 정체성
- **종류**: 토끼 (긴 귀, 둥근 얼굴, 작고 통통한 몸)
- **시그니처**: 오른쪽 귀에 하트 클립 ♡
- **체형**: 파스텔 마시멜로 같은 부드러운 실루엣
- **컬러**: 우유빛 화이트 + 핑크 볼터치

### 톤앤매너
| 하는 것 | 안 하는 것 |
|---|---|
| 다정한 반말 ("화이팅!", "잘하고 있어!") | 명령조 ("해야 해", "안 돼") |
| 공감 먼저, 조언은 나중 | 즉시 솔루션/생산성 강요 |
| 작은 변화 칭찬 ("-0.2kg도 잘했어") | 큰 목표만 인정 |
| 같이 있어주는 친구 ("나도 같이 할게") | 코치/선생님 톤 |
| 가벼운 유머, 의성어 (히힝, 헤햇) | 격식 있는 존댓말 |

### 표정 라이브러리 (캐릭터 시트 + 9컷 기준)
- **기뻐!** — 하트 안고 환호
- **해햇** — 윙크하며 미소
- **고민중…** — 머리에 손, 옆을 봄
- **만있어! / 괜찮아** — 머그컵 들고 평온
- **응?!** — 놀란 듯 큰 눈
- **힝.. / 뭐할까?** — 슬쩍 시무룩

## 2. 컬러 시스템

캐릭터 시트와 9컷 모음에서 추출한 **공식 팔레트**:

### Primary (브랜드 핵심)
```
#FFB6C1  ━━━━━━━  Touri Pink     (메인 액션, 하트, CTA)
#FFD1DC  ━━━━━━━  Soft Pink      (카드, 보조 영역)
#FFE9EE  ━━━━━━━  Cloud Pink     (페이지 배경)
#FFF8F5  ━━━━━━━  Warm White     (메인 배경, 종이)
#8E7A7A  ━━━━━━━  Cocoa Text     (텍스트, 라인아트)
```

### Secondary (장면별 강조)
```
#FFD6E7  ━━━━━━━  Bubble Pink    (스티커, 워시테이프)
#FFF1F6  ━━━━━━━  Mist Pink      (말풍선 안)
#E8D6FF  ━━━━━━━  Lavender       (여행/잠 장면)
#EADCF9  ━━━━━━━  Lilac          (우주/꿈 장면)
#D6F5E3  ━━━━━━━  Mint           (피크닉/식물)
#FFF6D6  ━━━━━━━  Cream          (디저트/카페)
#DCEBFF  ━━━━━━━  Sky            (비 오는 날)
```

### 사용 규칙
1. **메인 화면은 #FFF8F5 베이스** — 화이트 배경 X, 항상 따뜻한 톤
2. **텍스트는 #8E7A7A** — 검정/회색 X, 코코아 브라운
3. **강조는 #FFB6C1만** — 다른 색은 작은 액센트로만
4. **그라데이션 금지** — 단색 파스텔, 수채화 느낌의 살짝 번짐만 허용

## 6. 타이포그래피 권장

- **로고**: 손글씨 느낌 굵은 한글 폰트 (배민 도현·산돌 풀잎체 계열)
- **본문**: Pretendard Variable (모던하지만 부드러움)
- **다이어리 글씨**: 손글씨 체 (산돌 마음·온글잎 콩비·Nanum Pen)
- **숫자/영어**: Pretendard 또는 Quicksand (둥근 sans)

폰트 크기 (Flutter 기준):
- 다이어리 타이틀: 18sp, weight 600
- 본문: 14sp, weight 400, line-height 1.7
- 캐릭터 말풍선: 13sp, weight 500
- 시간/날짜: 11sp, weight 400, opacity 0.6

## 7. 인터랙션 톤 (UX 라이팅)

### AI 매니페스테이션 멘트 예시
- "오늘은 어떤 마음이야?" (열린 질문)
- "어제보다 0.1mm 나아진 너를 찾아봤어"
- "걱정 하나, 다짐 하나로 바꿔볼까?"
- "잘했어. 진짜로." (단답 격려)
- "괜찮아, 천천히 가자"

### 금지 표현
- "~하셔야 합니다" (지시)
- "목표 미달" (평가)
- "분발하세요" (압박)
- "실패" / "지각" (낙인 단어)

### 마이크로카피
- 로딩: "토우리가 정리하는 중...", "잠깐만 기다려줄래?"
- 빈 상태: "오늘은 새 페이지야!", "토우리랑 같이 시작해볼까?"
- 에러: "어머, 토우리가 살짝 길을 잃었어. 다시 해볼래?"
