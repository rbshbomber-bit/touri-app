# 🌌 토우리 키우기 — 장기 로드맵

> "별가루에서 시작해, 별의 화신이 되고, 별자리를 그리는 여정"
>
> 사용자가 30일 만에 마스터 찍고 끝나는 게 아니라, **무한 progression + 커뮤니티 확장**으로
> 1년 이상 붙들 수 있는 시스템 설계. Phase 1은 지금 구현됨. Phase 2/3는 비전 보존용.

---

## 🎯 핵심 철학

토우리는 **별에서 온 존재**다. 사용자의 행동(다이어리/매니페스트/감사/영성)이
"별을 부르는 행위" → 별가루가 모여 → 토우리 형성 → 자라남 → 빛남 → 화신 →
**다시 우주로 돌아가 별자리를 그림** → 무한 순환.

3축 시스템:
1. **개인 성장** (Phase 1, 종착) — 별가루 → 마스터
2. **무한 컬렉션** (Phase 2) — 별자리 시스템
3. **커뮤니티** (Phase 3) — 다른 사용자와 별 교환

---

## ✅ Phase 1 — 개인 성장 (구현 완료)

### 성장 5단계
| Stage | 이름 | XP | 도달 예상 |
|---|---|---|---|
| 0 | ✨ 별가루 | 0 | 시작 |
| 1 | 🐣 아기 토우리 | 5 | 첫 다이어리 |
| 2 | 🐰 친구 토우리 | 30 | ~3주 |
| 3 | 🌟 반짝 토우리 | 100 | ~3개월 |
| 4 | 👑 마스터 토우리 | 500 | ~1년 |

### 능력치 5종 + reward
| 행동 | 보상 |
|---|---|
| 다이어리 본문 10자+ | 💗 마음 +1/일 |
| Manifestation 저장 | ⭐ 반짝임 +1/일 |
| 감사 3줄+ | 💞 사랑 +1/일 |
| 뉴스 1편 | 🎯 집중 +1/일 *(미연결)* |
| 영성 체크 완료 | 🔥 용기 +1/일 *(미연결)* |
| ✦ 그려줘 성공 | ⭐ 반짝임 +2/회 |

### 일일 돌보기
밥주기 / 안아주기 / 재우기 — 일 1회, hunger/mood/energy +3.

### 출석 streak
매일 들어오면 +1, 빠지면 1로 리셋. 안 들어온 일수만큼 hunger/mood/energy 감소(최대 -3).

## 🌌 Phase 2 — 별자리 시스템 (마스터 이후, 무한)

### 컨셉
마스터 도달 → **"별의 화신"으로 승격** → 매일 행동마다 ⭐ **별** 획득 →
별 7개 모이면 **별자리 1개 완성** + 사용자가 이름 부여 → 무한 컬렉션.

### 별 획득표
| 행동 | 별 |
|---|---|
| 다이어리 1편 | ⭐ 1 |
| Manifestation | ⭐ 1 |
| 감사 3줄 | ⭐ 1 |
| 뉴스 1편 | ⭐ 1 |
| 영성 체크 완료 | ⭐ 1 |
| ✦ 그려줘 | ⭐ 2 |
| 연속 7일 보너스 | ⭐ 5 |
| 연속 30일 보너스 | ⭐ 20 |

### 별자리 (Constellation)
- 별 7개 모이면 별자리 1개 unlock
- 사용자가 별자리 이름 지정 (예: "첫 결심의 자리", "엄마의 자리", "프로젝트 토우리")
- 별자리 화면에서 모든 별자리 컬렉션 보기 (밤하늘 배경 + 빛나는 별자리들)
- 별자리에 메모 추가 가능 (이 별자리를 그릴 때 어떤 마음이었는지)
- 사용자 본인의 "전설" 만들기 — 1년이면 50+ 별자리

### 마스터의 5가지 길 (선택형)
마스터 도달 시 한 번만 선택 (변경 불가):
| 길 | 시그니처 능력 | 외형 |
|---|---|---|
| 💗 마음의 화신 | 다이어리 보상 ×2 | 핑크 후광 |
| 🎯 집중의 화신 | 뉴스 보상 ×2 | 보라 후광 |
| 💞 사랑의 화신 | 감사 보상 ×2 + 별 +1 | 골드 후광 |
| 🔥 용기의 화신 | 영성 보상 ×2 | 빨강 후광 |
| ⭐ 반짝임의 화신 | 그려줘 무료 (월 5회) | 무지개 후광 |

## 🤝 Phase 3 — 커뮤니티 (Phase 2 완료 후)

### 친구 토우리
- 사용자끼리 친구 등록
- 친구의 토우리 단계/별자리 일부 보기

### 별 선물
- 친구 토우리에게 ⭐ 별 1개 보내기 (하루 3명까지)
- 받으면 내 토우리 별자리 진행도 +1
- 친구가 어려운 일 겪고 있을 때 응원의 별 보내기

### 공동 별자리
- 친구 2명 이상이 함께 만드는 별자리
- 예: 커플 별자리 "우리의 결혼", 친구 별자리 "함께 시험 합격"
- 공동 별자리는 모든 참여자에게 표시

### 별의 전당 (Hall of Stars)
- 마스터 + 별자리 100개 이상 사용자 입성
- 다른 사용자들이 본받을 수 있는 익명 토우리 컬렉션 공개
- "내가 따라 하고 싶은 토우리" 즐겨찾기 가능
